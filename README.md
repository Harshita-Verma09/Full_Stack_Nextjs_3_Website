# Full Stack Next.js UI Assignment

##  Project Overview

This project is a **Next.js-based UI-focused assignment** where the primary goal was to design and implement a clean, responsive, and user-friendly interface. The emphasis was on **UI/UX, component structure, and layout**, rather than heavy backend logic.

The application demonstrates modern frontend development practices using **Next.js App Router**, reusable components, and responsive design techniques.
